## 执行方法：
### 克隆此代码库并将网站源码打成 `jar` 包，或直接下载已打包好的 `jar` 包
#### 下载代码并打包
```shell
# 下载代码库并进行打包
> git clone https://github.com/Leven0212/backup-system-web.git
> mv backup-system-web-master backup-system-web
> cd backup-system-web
> mvn clean package
```
